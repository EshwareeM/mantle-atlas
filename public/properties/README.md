# Property Images

This folder contains images for rental assets displayed on the Rental Assets page.

## Current Setup
- Currently using SVG placeholder images with colored backgrounds
- Images are referenced in `/data/rentalAssets.ts`

## To Replace with Your PNG Images
1. Place your PNG images in this folder with these exact names:
   - austin-loft.png
   - chicago-office.png  
   - denver-house.png
   - manhattan-apartment.png
   - miami-beach-condo.png
   - seattle-townhouse.png

2. Update the image URLs in `/data/rentalAssets.ts` to use `.png` instead of `.svg`
